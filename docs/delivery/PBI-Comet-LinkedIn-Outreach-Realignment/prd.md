# PRD: Comet LinkedIn Outreach Realignment

## 1. Problem Statement
The project’s outreach and GTM artifacts diverged over time across OpenClaw-oriented planning, earlier deep research, adversarial claim validation, and more recent Comet/LinkedIn recommendations. This creates risk that current conclusions, live-site messaging, and implementation priorities are no longer fully aligned with the strongest combined research direction.

## 2. Goals
1. Reconcile earlier research conclusions against current LinkedIn/Comet recommendations.
2. Identify where current conclusions drift from the strongest evidence-backed GTM path.
3. Re-anchor the outreach strategy on a small set of defensible SKUs, claims, and routes.
4. Update live docs and selected product surfaces so LinkedIn click-through matches the reconciled strategy.
5. Produce an implementation-backed gap analysis with clear high/medium/low priorities.

## 3. Scope
### In Scope
- Adversarial comparison across GTM, outreach, market-alignment, USP, Top20, and combined synthesis docs.
- Reconciliation of marketplace/channel recommendations with the full research set.
- Updates to canonical outreach strategy documents.
- Targeted copy/route/product changes needed to align stakeholder-facing claims with reconciled conclusions.
- Delivery artifacts documenting implementation status and next actions.

### Out of Scope
- Large new product modules not already justified by current codebase maturity.
- Full marketplace integrations for Microsoft, Snowflake, or AWS.
- CRM/provider integrations or outbound automation systems.

## 4. Success Metrics
- One clear canonical LinkedIn/Comet outreach strategy aligned with prior research.
- A documented agreement/disagreement matrix covering earlier vs current conclusions.
- High/medium action items mapped to implemented / partial / deferred status.
- Stakeholder-facing pages avoid claims contradicted by adversarial analysis.
- Next implementation plan focuses on the most defensible LinkedIn conversion path.
