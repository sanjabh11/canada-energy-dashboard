# Energy Corpus Seed Set

This directory is the first curated corpus for trustworthy energy-domain retrieval.

## Purpose
- Provide source-grounded reference material for future embedding and RAG work.
- Separate domain knowledge from implementation notes and developer docs.
- Keep each document focused on one policy, market, or regulatory topic.

## Corpus Rules
- Prefer official Canadian market operator, regulator, or government sources.
- Keep summaries factual and compact.
- Include source URLs so later ingestion jobs can attach provenance.
- Split long topics into multiple documents instead of creating one giant file.
