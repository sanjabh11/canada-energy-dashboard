# Audit Reports

This directory contains dated audit reports generated by the Fable 5-optimized repo audit prompt.

## How to generate an audit

### Using the Claude Code slash command:
```
/repo-audit
/repo-audit security
/repo-audit src/billing
```

### Using the standalone prompt:
Copy the contents of `prompts/fable5-repo-audit-prompt.md` into your AI tool.

## Audit naming convention
`audit-{YYYY-MM-DD}.md`
