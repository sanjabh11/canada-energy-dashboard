Invoke the `fable5-prompt` skill to perform a full 360-degree audit of this repository.

The skill is located at `~/.codeium/windsurf/skills/fable5-prompt/SKILL.md`.

Read the SKILL.md file and follow its instructions to:

1. **Phase 0**: Calibrate to the project's own config and standards
2. **Phase 0.5**: Discover available tooling (linters, test runners, security scanners)
3. **Phase 1**: Quantitative discovery, feature analysis, ICP development, TAM/SAM/SOM, and buyer journey mapping
4. **Phase 2**: Parallel subagent audit across 19 dimensions with adversarial verification and competitive intelligence per dimension
5. **Phase 3**: Improvement strategy with themes, target states, market alignment, and sales enablement artifact audit
6. **Phase 4**: Agent-executable task plan with M0–M4 milestones (requires approval)
7. **Phase 5**: Comprehensive success criteria verification including GTM readiness gate
8. **Phase 6**: Presales lifecycle and conversion funnel audit

Output: `docs/audits/audit-{YYYY-MM-DD}.md`

Optional focus (pass as argument):
- No argument: full 360-degree audit (all 19 dimensions, all phases)
- `security`: security-focused audit only
- `performance`: performance-focused audit only
- `src/billing`: audit specific subdirectory
- `auth`: audit specific concern across codebase
- `market`: marketability and seller proposition audit only
- `marketing`: marketing strategy, positioning, and messaging audit only
- `outreach`: outreach, engagement, and cadence audit only
- `pricing`: pricing strategy and monetization model audit only
- `presales`: presales lifecycle and conversion funnel audit only
- `gtm`: go-to-market readiness gate only
